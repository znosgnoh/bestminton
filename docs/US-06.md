# US-06: Enter Number of Guests per Member

**Epic:** Attendance & Input

**As a** team captain,
**I want to** enter the number of guests a member brought,
**so that** members who brought additional players share a larger portion of the cost.

## Acceptance Criteria

- [ ] Each checked member's row displays a "Guests" input field, defaulting to `0`.
- [ ] The field accepts non-negative integers only (0, 1, 2, …).
- [ ] Negative values are rejected with an inline error.
- [ ] A member with `guests = 1` has their weight doubled relative to a solo member with the same hours.
- [ ] The field is disabled (or hidden) for unchecked members.
