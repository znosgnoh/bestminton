# US-27: Enter Match Cost and Hours After the Game

**Epic:** Post-Match Settlement

**As a** captain,
**I want to** enter the total court cost, who paid, and how many hours were played after a match ends,
**so that** the app can calculate each player's share for Splitwise.

## Acceptance Criteria

- [ ] The match detail page (`/matches/[id]`) shows a "Settle Match" section for past matches.
- [ ] The captain can enter: Total Cost (THB), Paid By (dropdown of registered players), Hours Played.
- [ ] Submitting saves these values to the Match record via `PUT /api/matches/[id]`.
- [ ] Once cost and hours are saved, the calculated share per player is displayed in a review table.
- [ ] The review table shows: name, guests, weight, and owed amount for each registered player.
- [ ] The totals row confirms the sum equals Total Cost exactly (rounding fix applied).
- [ ] Fields are pre-filled if already saved (to allow corrections before syncing).
