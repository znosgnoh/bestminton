# US-14: Handle Member Fetch Failure

**Epic:** Error Handling & Edge Cases

**As a** team captain,
**I want to** see a helpful error if the member list cannot be loaded,
**so that** I understand why the page is not functional and what to do next.

## Acceptance Criteria

- [ ] If the Splitwise group member fetch fails, an error banner is displayed.
- [ ] The banner includes a "Retry" button to attempt the fetch again.
- [ ] The rest of the form is disabled until members are successfully loaded.
