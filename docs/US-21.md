# US-21: Management — Add a Team Member

**Epic:** Member Management

**As a** captain,
**I want to** add a player to the team registry on `/management`,
**so that** they appear on the match roster and can self-register for sessions.

## Acceptance Criteria

- [ ] A form on `/management` accepts: Name (required), Avatar URL (optional), Splitwise ID (optional integer).
- [ ] Submitting the form calls `POST /api/members` and saves the member to the database.
- [ ] The new member immediately appears in the member list on the page without a full reload.
- [ ] Name is required; submitting without a name shows an inline validation error.
- [ ] Splitwise ID must be a positive integer if provided; invalid input shows an error.
- [ ] Duplicate Splitwise IDs are rejected by the API with a 409 response and a user-facing error message.
