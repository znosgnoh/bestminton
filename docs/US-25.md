# US-25: Management — View and Edit Matches

**Epic:** Match Management

**As a** captain,
**I want to** see all scheduled matches and edit or delete them from `/management`,
**so that** I can correct mistakes or cancel sessions.

## Acceptance Criteria

- [ ] `/management` shows a list of all matches: title, venue, date/time, recurring badge if applicable.
- [ ] Each match row has an "Edit" button that opens an inline form pre-filled with match data.
- [ ] Saving edits calls `PUT /api/matches/[id]`; the list and the homepage update immediately.
- [ ] Each match row has a "Delete" button with a confirmation prompt.
- [ ] Deleting a match cascades: all related registrations and guests are deleted.
- [ ] Matches that have already been synced to Splitwise are marked with a "Synced ✓" badge and cannot be deleted without explicit confirmation.
