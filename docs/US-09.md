# US-09: Review Summary Before Sync

**Epic:** Cost Calculation & Review

**As a** team captain,
**I want to** see a clear summary of the session before I sync,
**so that** I can catch any mistakes without creating an incorrect Splitwise expense.

## Acceptance Criteria

- [ ] A "Review" section or modal shows: session total, payer name, and a table of all participating members with their owed amounts.
- [ ] The table includes a totals row confirming the sum equals `Total_Cost`.
- [ ] The captain can navigate back to edit attendance or amounts before confirming.
- [ ] No data is sent to Splitwise until the captain explicitly clicks "Sync to Splitwise".
