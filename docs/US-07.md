# US-07: Calculate Each Member's Owed Share

**Epic:** Cost Calculation & Review

**As a** team captain,
**I want to** see the exact amount each member owes,
**so that** I can verify the split is fair before pushing it to Splitwise.

## Acceptance Criteria

- [ ] For each checked member `i`, the app computes: `W_i = hours_i * (1 + guests_i)`.
- [ ] The app computes `W_total = Sum(W_i)` across all checked members.
- [ ] The app computes `Owed_i = Total_Cost * (W_i / W_total)`, rounded to two decimal places.
- [ ] The review table displays each member's name, hours, guests, weight, and owed share.
- [ ] The sum column footer confirms the total equals `Total_Cost` exactly.
