# US-20: Add a Guest to a Player Registration

**Epic:** Match Registration

**As a** registered player,
**I want to** add guests to my registration,
**so that** the app knows to split the cost fairly — guests share the cost with the player who brought them.

## Acceptance Criteria

- [ ] After a player registers, a "+ Guest" button appears next to their entry.
- [ ] Clicking "+ Guest" adds an unnamed guest slot linked to that player's `MatchRegistration`.
- [ ] The guest count is visible on the player's registration row (e.g. "+2 guests").
- [ ] Each guest can optionally be given a label (e.g. "Wife", "Friend") via an inline text input.
- [ ] A guest can be removed via an ✕ button; this calls `DELETE /api/matches/[id]/guests/[guestId]`.
- [ ] The weight formula accounts for guests: `W_i = hours * (1 + guestCount_i)`.
