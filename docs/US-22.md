# US-22: Management — Create a Single Match

**Epic:** Match Management

**As a** captain,
**I want to** schedule a one-off match on `/management`,
**so that** players can see it on the homepage and register.

## Acceptance Criteria

- [ ] A form on `/management` accepts: Title (required), Venue (required), Date (required), Time (required).
- [ ] The "Recurring" toggle is off by default.
- [ ] Submitting calls `POST /api/matches` and the match appears in the match list immediately.
- [ ] The new match shows up under "Upcoming" on the homepage.
- [ ] All required fields are validated before submission; errors are shown inline.
- [ ] Date must be in the future; past dates are rejected with an inline error.
