# US-18: View Upcoming vs Past Matches (Tabs)

**Epic:** Match Discovery

**As a** player,
**I want to** switch between "Upcoming" and "Past" match tabs on the homepage,
**so that** I can quickly find the next game or review a previous session.

## Acceptance Criteria

- [ ] The homepage has two tabs: "Upcoming" and "Past".
- [ ] "Upcoming" shows matches whose `scheduledAt` is in the future, sorted ascending.
- [ ] "Past" shows matches whose `scheduledAt` is in the past, sorted descending (most recent first).
- [ ] The active tab is highlighted.
- [ ] Tab state is preserved in the URL (e.g. `?tab=past`) so the link is shareable.
- [ ] "Upcoming" is the default tab.
