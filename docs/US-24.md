# US-24: Management — View and Edit Members

**Epic:** Member Management

**As a** captain,
**I want to** see all registered team members and edit or delete them,
**so that** the roster stays accurate over time.

## Acceptance Criteria

- [ ] `/management` shows a list of all members: avatar, name, and Splitwise ID (if set).
- [ ] Each member row has an "Edit" button that opens an inline form pre-filled with their current data.
- [ ] Saving edits calls `PUT /api/members/[id]`; the list updates immediately.
- [ ] Each member row has a "Delete" button.
- [ ] Deleting a member shows a confirmation prompt before calling `DELETE /api/members/[id]`.
- [ ] A member cannot be deleted if they have existing match registrations (API returns 409); a user-facing error is shown instead.
