# US-54: View Member Rankings Leaderboard

**Epic:** Leaderboard & History

**As a** player,
**I want to** see a leaderboard sorted by Elo,
**so that** I know who the top players are.

## Acceptance Criteria

- [ ] Leaderboard page lists all members: rank, name, Elo, W–L record, token balance.
- [ ] Sorted by `eloRating` descending; ties broken by `totalWins` then name.
- [ ] Accessible without login from main navigation.
- [ ] Empty state if no completed challenges yet (all at default 1200).
