# US-49: Enforce Mutual Exclusion Between Betting Columns

**Epic:** Betting Board (No-Login Flow)

**As a** player,
**I want** my name to be locked to one side only,
**so that** I cannot accidentally bet on both outcomes.

## Acceptance Criteria

- [ ] If Member X is checked in Column A, Column B shows X as disabled or auto-unchecked.
- [ ] Switching sides (check B after A) moves the bet atomically — never two active bets for the same bettor.
- [ ] Visual state makes the active side obvious (checked + other column greyed/disabled for that row).
