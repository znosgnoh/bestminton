# US-47: View Challenge Match Info (Elo, Probability, Handicap)

**Epic:** Challenge Creation & Match Info

**As a** player,
**I want to** see both competitors' ratings, win probability, and suggested handicap,
**so that** I know how fair the matchup is before betting.

## Acceptance Criteria

- [ ] Top section shows: Player A vs Player B (names/avatars), current Elo, win probability %, suggested handicap points.
- [ ] `E_A = 1 / (1 + 10^((R_B - R_A) / 400))` (and symmetric for B); displayed rounded to whole % or 1 decimal.
- [ ] `Handicap = round(abs(R_A - R_B) / 50)`; UI states which player **receives** the head-start in a 21-point game (lower Elo).
- [ ] Values load when the challenge screen opens; live recalculation if ratings change elsewhere is not required in v1.
- [ ] If both players have equal Elo, handicap shows `0`.
