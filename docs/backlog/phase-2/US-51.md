# US-51: Resolve Challenge and Select Winner

**Epic:** Match Resolution

**As a** admin,
**I want to** mark the match complete and pick the winner,
**so that** ratings and tokens update automatically.

## Acceptance Criteria

- [ ] "End Match & Resolve" visible only when status is `ACTIVE`.
- [ ] Action requires admin authorization (US-56).
- [ ] Admin selects winner: Player A or Player B (or Side A / Side B for doubles).
- [ ] Confirmation step before irreversible resolution.
- [ ] On success, status → `COMPLETED`; `winnerId` stored; `completedAt` set.
- [ ] UI shows final result and updated Elo/token balances (US-52, US-53).
