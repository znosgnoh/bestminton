# US-55: View Challenge History

**Epic:** Leaderboard & History

**As a** player,
**I want to** browse past and active challenges,
**so that** I can review results and bets.

## Acceptance Criteria

- [ ] List shows challenges with status badge (`PENDING`, `ACTIVE`, `COMPLETED`), players, date.
- [ ] Tapping a row opens the challenge detail screen (read-only for `COMPLETED`).
- [ ] `ACTIVE` challenges show locked betting board; `COMPLETED` show final pools and payouts.
- [ ] Default sort: newest first.
