# US-52: Update Elo Ratings on Challenge Completion

**Epic:** Match Resolution

**As a** system,
**I want** to recalculate Elo for both competitors when a challenge completes,
**so that** rankings reflect match outcomes.

## Acceptance Criteria

- [ ] Server-side only: `New_R = Old_R + K * (Actual - Expected)` where Actual = 1 (win) or 0 (loss).
- [ ] `K = 32` if `totalMatches < 10`, else `K = 16`.
- [ ] `totalMatches` increments for both players; `totalWins` increments for winner only.
- [ ] Ratings persist to DB as integers; UI shows before → after on the completed challenge screen.
- [ ] 2v2: each of the four players updated per US-44 rules.
