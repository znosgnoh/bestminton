# US-44: Support 2v2 Challenges

**Epic:** Data Foundation

**As a** captain,
**I want to** create a doubles challenge with two players per side,
**so that** pairs can compete and bet the same way as singles.

## Acceptance Criteria

- [ ] Challenge creation allows selecting 2 players for Side A and 2 for Side B (4 distinct members).
- [ ] Side Elo for display/calculation uses the **average** of the two players' ratings (documented in UI).
- [ ] Handicap and win probability use the averaged ratings.
- [ ] On completion, **both** players on the winning side receive a win; **both** on the losing side receive a loss for `totalMatches` / `totalWins`.
- [ ] Elo updates apply individually per player using the same K-factor rules against the opponent side's average rating.
