# US-48: Place Bets via Two-Column Checkbox Board

**Epic:** Betting Board (No-Login Flow)

**As a** group member,
**I want to** tick my name under the player I'm backing,
**so that** I can join the betting pool without logging in.

## Acceptance Criteria

- [ ] Middle section renders two columns: "Bet on Player A" and "Bet on Player B".
- [ ] Each column lists **all** group members with a checkbox.
- [ ] Checking a name creates/updates a bet for that member on the selected side (amount = 1 token).
- [ ] Unchecking removes the bet while challenge is `PENDING`.
- [ ] Each column shows a running count: "N bets · N tokens in pool".
- [ ] Competitors (Player A / B) may bet on themselves or others — no restriction unless product later forbids it.
