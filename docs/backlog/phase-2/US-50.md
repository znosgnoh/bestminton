# US-50: Lock Bets When Admin Starts the Challenge

**Epic:** Match Resolution

**As a** admin,
**I want to** lock all bets before play begins,
**so that** no one changes wagers mid-match.

## Acceptance Criteria

- [ ] "Start Challenge (Lock Bets)" is in the bottom admin section, visible only when status is `PENDING`.
- [ ] Action requires admin authorization (US-56).
- [ ] On success, status → `ACTIVE`; all checkboxes disabled; pools frozen.
- [ ] No new bets or bet removals allowed while `ACTIVE`.
- [ ] Button shows loading/disabled state during API call; errors surface clearly.
