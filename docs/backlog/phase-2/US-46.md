# US-46: Create a New Challenge

**Epic:** Challenge Creation & Match Info

**As a** player on a shared device,
**I want to** start a new challenge by picking the competitors,
**so that** the group can bet before the game begins.

## Acceptance Criteria

- [ ] A "New Challenge" entry point is reachable from the app nav or a dedicated `/challenges` route.
- [ ] User selects Player A and Player B from the member list (avatars/names, same roster as Phase 1).
- [ ] A player cannot be selected on both sides.
- [ ] On submit, a `PENDING` challenge is created and the user lands on the challenge screen (US-47).
- [ ] Validation error if fewer than 2 distinct players are selected.
