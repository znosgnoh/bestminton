# US-05: Enter Playing Hours per Member

**Epic:** Attendance & Input

**As a** team captain,
**I want to** enter the number of hours each participating member played,
**so that** members who played longer contribute a proportionally larger share.

## Acceptance Criteria

- [ ] Each checked member's row displays an "Hours" input field.
- [ ] The field accepts positive numeric values (e.g., 1, 1.5, 2).
- [ ] Zero or negative values are rejected with an inline error.
- [ ] Hours is a required field for any checked member.
- [ ] The field is disabled (or hidden) for unchecked members.
