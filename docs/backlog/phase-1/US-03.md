# US-03: Fetch Group Members from Splitwise

**Epic:** Attendance & Input

**As a** team captain,
**I want to** automatically load the list of members from my Splitwise group,
**so that** I don't need to manually enter names and can avoid mismatches with Splitwise IDs.

## Acceptance Criteria

- [ ] On page load (or on a "Load Members" action), the app calls the Next.js proxy, which fetches members from the Splitwise API.
- [ ] A loading indicator is shown while the fetch is in progress.
- [ ] Each member's name is displayed in a list with a checkbox.
- [ ] If the API call fails, a clear error message is shown and the user can retry.
- [ ] The list is not empty after a successful fetch.
