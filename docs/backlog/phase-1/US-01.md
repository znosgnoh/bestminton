# US-01: Enter Total Session Cost

**Epic:** Session Initialization

**As a** team captain,
**I want to** enter the total court fee for the session,
**so that** the app knows the fixed amount to split among participants.

## Acceptance Criteria

- [ ] A numeric input field labeled "Total Cost" is displayed prominently on the page.
- [ ] The field accepts only positive numeric values (integers or decimals).
- [ ] The field rejects zero, negative numbers, and non-numeric characters, showing an inline error message.
- [ ] The field is required — proceeding without it is blocked.
