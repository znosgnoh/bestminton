# US-26: View Match Detail Page

**Epic:** Match Registration

**As a** player,
**I want to** open a match detail page from the homepage,
**so that** I can see who is joining and register myself.

## Acceptance Criteria

- [ ] Tapping a match card on the homepage navigates to `/matches/[id]`.
- [ ] The page shows: title, venue, date/time, registered players with their guest counts.
- [ ] The full member roster is shown as avatar tiles so players can tap to register.
- [ ] For past matches, the page shows the final registrations and the calculated split (if cost was entered).
- [ ] The page is shareable — opening the URL directly shows the correct match data.
- [ ] If the match ID does not exist, a 404 page is shown.
