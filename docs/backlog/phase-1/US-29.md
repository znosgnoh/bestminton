# US-29: Add Guest with Playtime Flag at Creation

**Epic:** Guest Management

**As a** player,
**I want to** set a guest's name and playtime (Full / Half time) when adding them,
**so that** I don't have to toggle it separately after the guest appears.

## Acceptance Criteria

- [x] Clicking **+ Guest** opens an inline card form (not a modal) with two fields: Name (optional text input) and a Full / ½ time toggle (default: Full).
- [x] The Full button is highlighted emerald when selected; the ½ time button is highlighted amber when selected.
- [x] Pressing Enter in the name field submits the form; pressing Escape cancels it.
- [x] After a guest is added the form closes, the guest pill appears immediately, and the `+ Guest` button returns.
- [x] The `playedFull` value selected in the form is persisted to the database on creation.
- [x] The guest playtime can still be toggled (F / ½ button on the pill) after creation.
- [x] The form is only shown for upcoming matches (`!isPast`).
