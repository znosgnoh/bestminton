# US-30: Management Page Hidden from Navigation

**Epic:** Access Control

**As a** captain,
**I want** the management page to be accessible only by typing its URL directly,
**so that** casual players don't accidentally stumble into admin functions.

## Acceptance Criteria

- [x] The "Manage" link is removed from the site header.
- [x] `/management` remains fully functional when accessed by typing the URL directly.
- [x] All management features (member CRUD, match CRUD, settle) continue to work as before.
- [x] No authentication is required — security is by obscurity of the URL (captain shares the URL out-of-band).
