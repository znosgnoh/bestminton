# US-17: View List of Matches on Homepage

**Epic:** Match Discovery

**As a** player,
**I want to** see all scheduled matches on the homepage,
**so that** I know when and where the next session is and can plan to join.

## Acceptance Criteria

- [ ] The homepage (`/`) displays a list of match cards fetched from the database.
- [ ] Each card shows: title, venue, date/time, number of registered players, and match status (upcoming / past).
- [ ] Cards are sorted chronologically — soonest upcoming first, most recent past first.
- [ ] The page works as a Server Component; data is fetched server-side via Prisma.
- [ ] If there are no matches, an empty-state message is shown per tab.
