# US-23: Management — Create a Recurring Weekly Match

**Epic:** Match Management

**As a** captain,
**I want to** create a match series that repeats every week on the same day,
**so that** I don't need to manually create the same match every week.

## Acceptance Criteria

- [ ] The match creation form has a "Recurring — repeat every week" toggle.
- [ ] When toggled on, the day-of-week is derived from the selected date automatically.
- [ ] Enabling recurring stores `isRecurring = true` and `recurDayOfWeek` on the Match record.
- [ ] The system generates the next N occurrences (e.g. 4 weeks ahead) automatically when the recurring match is created.
- [ ] Each generated occurrence is an independent Match row (same title/venue/day, different `scheduledAt`).
- [ ] Recurring matches are visually labelled (e.g. a "↻ Weekly" badge) on the homepage and management list.
