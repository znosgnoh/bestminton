# US-28: Sync Match Expense to Splitwise

**Epic:** Post-Match Settlement

**As a** captain,
**I want to** sync the calculated expense to Splitwise from the match detail page,
**so that** members are notified of what they owe without manual entry in Splitwise.

## Acceptance Criteria

- [ ] The "Sync to Splitwise" button is only visible after Total Cost and Hours have been saved (US-27).
- [ ] The button is disabled when `SPLITWISE_API_KEY` / `SPLITWISE_GROUP_ID` are not configured; an info message explains why.
- [ ] The button is disabled if any registered player does not have a Splitwise ID set; a warning lists the missing members.
- [ ] Clicking the button calls `POST /api/splitwise/expense` with the calculated shares.
- [ ] On success, the match record is updated to `synced = true`; the button is replaced by a "Synced ✓" badge.
- [ ] On failure, a clear error message is shown and the button remains active for retry.
- [ ] Once synced, the sync button is permanently disabled for that match to prevent duplicate expenses.
